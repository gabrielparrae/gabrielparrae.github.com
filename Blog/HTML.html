HTML

¿Qué es el HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web. Sirve como el pilar fundamental de la web, definiendo la estructura básica de un documento mediante etiquetas y elementos. Wikipedia

Versiones de HTML y sus características principales

  1. HTML 2.0 (1995)

    2. Fue la primera especificación oficial de HTML publicada por el W3C.

    Estableció un estándar básico para el diseño de páginas web.

    Incluía elementos básicos como encabezados, párrafos, listas, enlaces, imágenes, y formularios simples.

    Limitaciones en estilos y multimedia. Se centraba más en la estructura del contenido que en la presentación.

  2. HTML 3.0 (1997)

    3. Introdujo capacidades más avanzadas como tablas, scripts y estilos más complejos.

    Sin embargo, fue difícil de implementar debido a las limitaciones de los navegadores de la época (como Netscape Navigator).

    Esta versión fue descontinuada rápidamente porque no tuvo soporte adecuado en el ecosistema.

  3. HTML 3.2 (1997)

    4. Versión revisada de HTML 3.0 con mejor soporte para navegadores.

    Añadió soporte para applet (Java), tablas y scripts, consolidando características existentes.

    Fue un estándar más exitoso y ampliamente utilizado.

    Permitió mayor flexibilidad para la creación de sitios web más dinámicos.

  4. HTML 4.0 (1999)

    5. Una actualización importante en la evolución de HTML.

    Introdujo el concepto de Document Object Model (DOM) y un mejor soporte para CSS para separar contenido y presentación.

    Agregó soporte para internacionalización (lenguajes y codificación de caracteres).

    Se dividió en tres versiones: Strict, Transitional y Frameset:

    Strict: Se enfocaba en la estructura y semántica del contenido, sin elementos obsoletos.

    Transitional: Permitía la transición de sitios web antiguos a las nuevas especificaciones.

    Frameset: Permitía la creación de sitios web con marcos (frames).
