HTML

¿Qué es el HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web. Sirve como el pilar fundamental de la web, definiendo la estructura básica de un documento mediante etiquetas y elementos. Wikipedia

Versiones de HTML

HTML 2.0

La versión 2.0 de HTML fue publicada en 1995 y estableció las bases del lenguaje que conocemos hoy en día.

HTML 3.0 y 3.2

HTML 3.0 fue una versión propuesta en 1997, pero nunca fue adoptada oficialmente. HTML 3.2, sin embargo, fue publicado en 1997 y añadió soporte para tablas y otros elementos.

HTML 4.0

HTML 4.0 fue publicado en 1997 y trajo consigo muchas mejoras, incluyendo soporte para hojas de estilo en cascada (CSS) y una mejor accesibilidad.

Elementos que componen un archivo HTML

Un archivo HTML típico incluye elementos como <html>, <head>, <title>, <body>, entre otros, se dejara un link con mas elementos.
